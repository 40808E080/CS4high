```
OpenCV 3是一種先進的電腦視覺庫，可以用於各種圖像和視頻處理操作，通過OpenCV 3 能很容易地實現一些有前景且功能先進的應用
（比如：人臉識別或目標跟蹤等）。理解與電腦視覺相關的演算法、模型以及OpenCV 3 API背後的基本概念，
有助於開發現實世界中的各種應用程式（比如：安全和監視領域的工具）。

本書將從影像處理的基本操作出發，帶你開啟先進電腦視覺概念的探索之旅。
電腦視覺是一個快速發展的學科，在現實生活中，它的應用增長得非常快，
因此寫作本書的目的是為了説明電腦視覺領域的新手和想要瞭解全新的OpenCV 3.0.0的電腦視覺專家。

本書的主要內容

第1章介紹如何在不同平臺下安裝基於Python的OpenCV，並給出一些常見問題的解決方法。

第2章介紹了OpenCV的I/O功能，並討論與項目相關的概念，以及如何針對該項目進行物件導向設計。

第3章介紹一些圖像變換方法，例如在圖像中檢測膚色、銳化圖像、標記主體輪廓，以及使用線段檢測器檢測人行橫道等。

第4章介紹如何利用深度攝像頭的資料來識別前景和背景區域，這樣就可以限制針對前景或背景的效果。

第5章介紹一些OpenCV的人臉檢測功能和相關的資料檔案，這些檔定義了跟蹤目標的特定類型。

第6章介紹如何用OpenCV來檢測圖像特徵，並利用這些特徵來匹配和搜索圖像。

第7章介紹目標檢測和目標識別的概念，這是電腦視覺中最常見的問題之一。

第8章對目標跟蹤進行深入探討，目標跟蹤是對攝像機中的圖像或視頻中移動的物體進行定位的過程。

第9章介紹基於OpenCV的人工神經網路，並介紹其在現實生活中的應用。

閱讀前的準備工作

本書第1章會指導讀者安裝所有必要軟體，你只需準備一台較新的電腦。另外，強烈推薦為電腦安裝攝像頭，但這並不是必備的。

本書的讀者對象
本書針對具有一定Python工作經驗的程式師以及想要利用OpenCV庫研究電腦視覺課題的讀者。
本書不要求讀者具有電腦視覺或OpenCV經驗，但要具有程式設計經驗。
```
